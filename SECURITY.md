# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Use GitHub's private vulnerability reporting instead:
👉 [Report a vulnerability](https://github.com/jaelving/xemd/security/advisories/new)

You can expect a response within 7 days. If a vulnerability is confirmed, a fix will be prioritised and a security advisory published once resolved.
